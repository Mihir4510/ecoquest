 Project Report
College Eco-Tracker (Green Campus Project) 🌱
1. Title

College Eco-Tracker (Green Campus Project) – A Web Portal for Tracking Campus Sustainability Activities

2. Objective

To build a web portal where students can log their eco-friendly activities like tree plantation, cycling, and waste recycling.

To calculate Green Points for each activity and motivate students through a Leaderboard System.

To help the college track total eco-friendly contributions (like total trees planted, CO₂ saved, etc.).

To promote environmental awareness and create a competitive, fun way of encouraging sustainability.

3. Scope

Students can register/login, submit eco-friendly activities, and check their scores on the leaderboard.

Admins (faculty/college staff) can verify student entries, manage activity categories, and see total reports.

Can be extended to:

Show CO₂ savings through graphs.

Integration with mobile apps.

Rewards system (certificates, badges).

4. Modules
🔹 1. User Module (Students)

Register/Login

Add eco-friendly activity (tree planting, cycling, waste management, etc.)

Earn Green Points based on activity type

View personal progress

🔹 2. Admin Module

Verify activities submitted by students

Manage activity categories and points

View statistics (total activities, top students, CO₂ savings)

🔹 3. Leaderboard Module

Rank students based on total points

Show Top 10 eco-friendly students

Display achievements

🔹 4. Analytics/Dashboard

Graphs & charts of activities (weekly/monthly)

CO₂ emission savings calculations

Campus-wide green statistics

5. Technology Used
🔹 Frontend

HTML, CSS, JavaScript → For basic structure & styling

React.js → For interactive user interface

🔹 Backend

Node.js + Express.js → To handle requests, APIs, and server logic

🔹 Database

MongoDB → To store student details, activities, and points

🔹 Other Tools

Chart.js / Recharts → To display graphs & analytics

JWT Authentication → For secure login & sessions

GitHub → Version control

VS Code → Development environment

6. Working of System (Flow)

Student signs up → logs into the portal.

Student adds an activity (e.g., “Planted 2 trees”).

System assigns Green Points → saves data in MongoDB.

Admin verifies activity → points confirmed.

Leaderboard updates → Top students displayed.

Dashboard shows overall campus contributions.

7. Advantages

Encourages eco-friendly behavior among students.

Creates a competitive but fun environment through leaderboard & points.

Provides college administration with useful data for reporting.

Unique project idea → not a typical CRUD app.

Can be expanded into a real initiative in your campus.

8. Future Enhancements

Add mobile app version.

Integrate QR codes for quick activity logging.

Auto-tracking (cycling/walking via GPS).

Add reward system (certificates, digital badges).

9. Conclusion

The College Eco-Tracker is an innovative web portal that helps students and the college contribute to sustainability. By gamifying eco-friendly activities with points & leaderboards, it motivates students while also tracking environmental impact. This project not only showcases technical skills in MERN stack development but also connects with real-world sustainability goals.
