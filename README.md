# EcoQuest 🌱
Generative AI-powered Sustainability Tracker for Campuses

## 🚀 Overview
EcoQuest motivates students and institutions to adopt eco-friendly habits.  
Users log their sustainable actions (like cycling to college or reusing bottles), earn points, and climb leaderboards.  
Institutions get real-time dashboards and reports to track their eco-initiatives.

## ✨ Current Features (MVP)
- User authentication & profile
- Log eco-friendly actions
- Assign eco-points to actions
- Leaderboard with top contributors
- Dashboard with basic eco-stats

## 🤖 Upcoming AI Features (Planned)
- **AI Action Classifier**  
  Users log actions in natural language (*"cycled to college"*) → AI classifies and assigns points automatically.

- **Eco-AI Mentor (Chatbot)**  
  Personalized chatbot for sustainability queries (*"How to save water in hostel?"*).  

- **AI Reports**  
  Generate data-driven sustainability reports with insights and recommendations.

## 🛠️ Tech Stack
- **Frontend:** React.js + Tailwind CSS  
- **Backend:** Node.js + Express.js  
- **Database:** MongoDB  
- **AI Integration (Coming Soon):** OpenAI GPT API for classification, chatbot, and reports
