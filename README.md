 # 🌱 EcoQuest

**EcoQuest** is a web application designed to help **college students track and participate in eco-friendly activities**. It motivates users to adopt sustainable practices and provides **data-driven insights** for colleges to monitor green initiatives.

---

## 🚀 Features
- Track sustainable actions like recycling, energy saving, and green initiatives  
- Reward system to motivate participation  
- College dashboard for monitoring eco-activities  
- Data-driven reports and analytics  
- Simple and user-friendly web interface  

---

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Additional Libraries:** EJS, Express-EJS-Layouts, bcrypt, dotenv  

---

## 🎯 Purpose
- Encourage college students to adopt eco-friendly habits  
- Make green initiatives fun, engaging, and trackable  
- Provide data insights to colleges for sustainability programs
